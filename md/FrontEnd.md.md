# FrontEnd-Dev

## What’s difference between Internet & Server?
It is a wire that connects multiple devices. And when millions of people need to access 24 hours non-stop, in order to handle that, server shows up. 

Server can be regarded as 24 hours giant library for people to access (Google, Facebook, …). And the device who’s asking for information, is called Client, and Google is server (Client->ISP(internet service provider)->DNS (Domain Name System)->Server). ISP, for example T-mobile, China Mobile. 



## Difference between Website & Web App?
Website is focus on reading information, while Web App is focus on interaction with users. 



## What is HTML / CSS / JS?
Three files that determine how the website is working. 

And react, angular, view, those are the javascript framework, to allow user interface with application. 

The server or database will sent back the HTML file for frontend. 



## What is React?
Free, open-source JavaScript library. Specifically for frontend development. 

In JavaScript, we used to have HTML + CSS + JS file for frontend dev, but while using React, it’s like putting 3 things combined together. 


## What you use Google OAuth?
Proof of a real human — Google has already verified the email, possibly enforced 2FA, and detected suspicious logins. You're borrowing their security infrastructure for free.
No password liability — you never store passwords, so a breach of your database doesn't leak credentials.
A stable user ID — the sub claim in the token is a unique, permanent identifier for that user.
